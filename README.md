Billwise - Smart Expense Splitter (Full-stack)
---------------------------------------------

How to run:

1) Backend (requires Node.js, MongoDB running locally or Atlas):
   - cd billwise/backend
   - copy .env.example to .env and set MONGO_URI and JWT_SECRET
   - npm install
   - npm run dev   (or npm start)

2) Frontend:
   - cd billwise/frontend
   - npm install
   - npm start

Frontend runs on http://localhost:3000 and backend on http://localhost:5000 by default.
